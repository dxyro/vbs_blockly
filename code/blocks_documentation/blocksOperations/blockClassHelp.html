Block Class

Category: Flowbot Operations

Image

Definition

Essential required block with the appropriate OOP (Object-Oriented-Programming) code structure, based in the "script support" library. This block has to contain the others blocks inside, no block should be outside this block. Only one blockClass should exist in the workspace. Throws a warning if it does not find a main block in the logic booth.

Parameters

  • name (string)

    • The string for the class name of code in VBScript, this string given is camelized for adapt to the VBS code.

  • environment (selectable)

    • Local/Production option for run the code that traduces in true for local option and false por production option

  • elements (blocks)

    • SapElement blocks to define the fields to inract with in SAP.

  • parameters (blocks)

    • SapParameter blocks to define object environment to the script execution.
