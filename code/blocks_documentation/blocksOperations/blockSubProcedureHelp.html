Block Sub Process

Category: Operations

Image

Definition

Block to define a sub process in VBScript, a sub process is like a function with the difference that the sub procedure doesn't return anything, this fragment of code also can be executed multiples times without repeat the logic.

Parameters

  • name (string)

    • The string for the sub procedure name of code in VBScript, this string given is camelized for adapt to the VBS code.

  • actions (blocks)

    • This booth has to contain content blocks that defined the logic than can called afeter several times.

Connections

  • Before:

    • Any, There are no connection restrictions.

  • After:

    • Any, There are no connection restrictions.
