Block Sap Abort Script

Category: Sap Actions

Image

Definition

Sends the instruction for stop the execution of VBScript and define the type of error that cause it.

Parameters

  • type (selectable)

    • Infra/bussines/blocked/unknown option to define type error for stop the script.

  • message (blocks)

    • Block that returns string value, define the message of stop process.

Connections

  • Before:

    • Any, There are no connection restrictions.

  • After:

    • Any, There are no connection restrictions.
